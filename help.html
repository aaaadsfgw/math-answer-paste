ヘルプ

基本の使い方

  1. 問題文を入力、またはブラウザで選択中のテキストを読み込みます。
  2. 出力モードを選び、「解く」を押します。
  3. 表示された答えだけ1行の結果をコピーして使います。

ショートカットで解く

  1. Webページ上の問題文をドラッグして選択します。
  2. Ctrl + Shift + Y を押します。
  3. ページ右上に小さく「解析中...」と表示されます。
  4. 解析が終わると「コピー完了: 答え」と表示され、最終回答だけがクリップボードに入ります。
  5. 必要な場所で Ctrl + V を押して貼り付けます。

右上のトースト表示は補助的な小さい通知です。通常のページ閲覧を邪魔しないよう、数秒後に自動で消えます。

Ctrl + Shift + C はChrome DevTools等と衝突しやすいため、標準ショートカットは Ctrl + Shift + Y にしています。

ショートカットが反応しない場合は、Chromeの chrome://extensions/shortcuts で割り当てを確認してください。

Ollamaの起動確認

ollama serve
ollama pull qwen3:8b
ollama run qwen3:8b

よくあるエラー

  • 接続できない場合は Ollama が起動しているか確認してください。
  • モデルが見つからない場合は `ollama pull モデル名` を実行してください。
  • 重い場合は `qwen3:8b` と `/no_think` の組み合わせを試してください。
  • 特定ページで選択テキストを取得できない場合は、popup画面に手動で貼り付けてください。

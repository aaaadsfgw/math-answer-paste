ヘルプ

基本の使い方

  1. 問題文を入力、またはブラウザで選択中のテキストを読み込みます。
  2. 出力モードを選び、「解く」を押します。
  3. 表示された `[ ]` 形式の結果をコピーして使います。

Ollamaの起動確認

ollama serve
ollama pull deepseek-r1:14b
ollama run deepseek-r1:14b

よくあるエラー

  • 接続できない場合は Ollama が起動しているか確認してください。
  • モデルが見つからない場合は `ollama pull モデル名` を実行してください。
  • 重い場合は `deepseek-r1:8b` や `qwen3:8b` へ切り替えてください。

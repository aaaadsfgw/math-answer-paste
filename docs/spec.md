# システム仕様書

## 目的

Math Answer Paste は、数学・情報系の問題文をローカルLLMで解析し、最終回答を `[ ]` 形式でコピーできる Chrome 拡張です。

## 画面

- ポップアップ: 問題入力、選択テキスト読み込み、モード選択、解析、結果表示、コピー
- 履歴: 過去の問題、回答、日時、モードを表示し、全削除できる
- 設定: モデル名、Ollama API URL、既定モード、デモモード、履歴保存を変更できる
- 例題: 対応例を表示し、クリックでポップアップに渡す
- ヘルプ: 使い方、Ollama起動方法、エラー対応を説明する
- About: システム概要、ローカルAIを使う理由、注意事項を説明する

## 機能

- 問題文入力
- 選択テキスト取得
- 出力モード切り替え
- Ollama API呼び出し
- デモ回答
- 回答抽出
- クリップボードコピー
- 履歴保存
- 設定保存

## データ保存

`chrome.storage.local` に次を保存します。

- `settings`: モデル名、API URL、既定モード、デモモード、履歴保存
- `history`: 問題、回答、詳細、モード、日時
- `pendingQuestion`: 例題画面からポップアップへ渡す一時的な問題文

## Ollama連携

`js/ollama-client.js` が `POST http://localhost:11434/api/chat` を呼びます。

```json
{
  "model": "qwen3:8b",
  "stream": false,
  "messages": [
    { "role": "system", "content": "回答形式を縛る指示" },
    { "role": "user", "content": "問題文" }
  ]
}
```

## 出力形式

答えだけモードでは説明なしで `[ ]` のみ返します。途中式・解説モードでは最後の行に `最終回答: [答え]` を含めます。ヒントモードでは最後の行に `最終回答: [ヒントのみ]` を含めます。

## デモモード

Ollamaが使えない環境でも動作確認できるよう、固定例を `js/demo-solver.js` で処理します。未対応の問題は `[デモ未対応]` を返します。

# システム仕様書

## 目的

Math Answer Paste は、数学・情報系の問題文をローカルLLMで解析し、答えだけを1行で表示・コピーできる Chrome 拡張です。

## 画面

- ポップアップ: 問題入力、選択テキスト読み込み、モード選択、解析、最終回答表示、コピー
- 履歴: 過去の問題、回答、日時、モードを表示し、全削除できる
- 設定: モデル名、Ollama API URL、既定モード、デモモード、履歴保存を変更できる
- 例題: 対応例を表示し、クリックでポップアップに渡す
- ヘルプ: 使い方、Ollama起動方法、エラー対応を説明する
- About: システム概要、ローカルAIを使う理由、注意事項を説明する

## 機能

- 問題文入力
- 選択テキスト取得
- 出力モード切り替え
- Ollama API呼び出し
- デモ回答
- 最終回答抽出
- クリップボードコピー
- 履歴保存
- 設定保存

## データ保存

`chrome.storage.local` に次を保存します。

- `settings`: モデル名、API URL、既定モード、デモモード、履歴保存
- `history`: 問題、回答、詳細、モード、日時
- `pendingQuestion`: 例題画面からポップアップへ渡す一時的な問題文

## Ollama連携

`js/ollama-client.js` が `POST http://localhost:11434/api/chat` を呼びます。過去会話は送らず、毎回現在の問題だけを単発で処理します。

```json
{
  "model": "qwen3:8b",
  "stream": false,
  "messages": [
    { "role": "system", "content": "回答形式を縛る指示" },
    { "role": "user", "content": "/no_think\n問題：2x + 3 = 11" }
  ]
}
```

## 出力形式

最終回答は答え単体のみです。角括弧は使いません。

- 数値: `4`
- 変数: `x=4`
- 複数解: `2,3`
- 選択肢: `ウ`
- 不明: `不明`

answerモードでは、AIに「答えだけを1行で返せ。説明、途中式、記号装飾は禁止。不明なら不明。選択肢問題なら記号のみ。」と指示します。

steps / explain モードでは途中式や説明を許可しますが、最後の行に `最終回答: 答え` を書かせます。hintモードでは答えを出さず、最後の行を `最終回答: ヒントのみ` にします。

## 最終回答抽出

`js/answer-parser.js` は次の順で処理します。

1. `<think>...</think>` を除去する
2. 空行を除く
3. 最後の空でない1行を最終回答として扱う
4. `最終回答:` または `最終回答：` が含まれる場合は、その後ろを答えとして扱う
5. 先頭と末尾の空白を削除する
6. 返答が空なら `不明` にする
7. AIが `[x=4]` のように返した場合は、外側の角括弧を外して `x=4` にする

## デモモード

Ollamaが使えない環境でも動作確認できるよう、固定例を `js/demo-solver.js` で処理します。未対応の問題は `デモ未対応` を返します。
